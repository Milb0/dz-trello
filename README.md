# Kanban Board Application

This project is a web-based **Kanban Board application** developed using **React** and **Vite**.  
It provides a visual task management system that helps users organize tasks into boards and cards.

## Project Description

The Kanban Board allows users to create multiple boards representing different stages of a workflow.  
Each board contains cards that represent tasks, which can be moved between boards using drag-and-drop functionality.

Cards support additional details such as descriptions, labels, and task checklists.  
The application uses browser-based storage to ensure data persistence across sessions.

---

## Features

- Create, rename, and delete boards
- Add, edit, and delete cards
- Drag and drop cards between boards
- Labels with customizable colors and text
- Task checklists within cards
- Modal interface for editing card details
- Persistent data storage using `localStorage`
